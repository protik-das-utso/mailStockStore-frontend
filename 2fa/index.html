2FA Code Generator

Convert your 2FA Secret (the long base32 string from Google Authenticator setup) into live 6-digit codes.

How to use

  1. Go to My Emails page and find the account you purchased
  2. Copy the 2FA Secret or 2FA Authenticator Code from the credentials (usually a long string like "JBSWY3DPEHPK3PXP")
  3. Paste it into the field below
  4. Your 6-digit login code appears automatically and updates every 30 seconds
  5. Use this code when logging into the email account

Paste the secret key (base32 string) — spaces and case don't matter.

Privacy note: All computation happens in your browser — no secrets are sent to our server.
